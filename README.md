# Nutricion BioWellness

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/LEO3347/nutricion)

Sitio web estatico para promocionar la carrera de Nutricion y presentar la institucion DIF Municipal Tapachula.

## Despliegue en Render

El proyecto incluye `render.yaml` para crear un Static Site en Render.

- Build command: ninguno
- Publish directory: `.`
- Entry point: `index.html`
